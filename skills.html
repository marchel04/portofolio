Teknologi & Tools

HTML

HTML adalah bahasa standar yang digunakan untuk membuat halaman web. HTML menggunakan tag untuk menentukan struktur dan konten, seperti judul, paragraf, gambar, dan tautan.

CSS

CSS digunakan untuk mengatur tampilan halaman web. CSS memungkinkan pengembang untuk menentukan warna, font, tata letak, dan elemen visual lainnya, sehingga membuat tampilan halaman web lebih menarik dan mudah dibaca.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaksi dan fungsionalitas ke halaman web. JavaScript dapat digunakan untuk membuat efek animasi, memvalidasi formulir, mengontrol elemen halaman, dan banyak lagi.

Bootstrap

Bootstrap adalah framework CSS open-source yang menyediakan berbagai komponen dan gaya yang siap pakai untuk membangun tampilan web yang responsif (tampilan yang menyesuaikan dengan berbagai ukuran layar). Bootstrap memudahkan pengembang untuk membuat desain web yang konsisten dan efisien.

Figma

Figma adalah alat desain berbasis web yang digunakan untuk membuat prototipe (model awal) dan desain antarmuka pengguna (UI) untuk aplikasi web dan seluler. Figma membantu desainer untuk merancang tampilan visual sebelum diimplementasikan dalam kode.
